## 🚀 Deploy on GitHub Pages (Free Hosting)

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) → Sign in / Sign up
2. Click **"New repository"**
3. Name it: `jivan-foundation`
4. Set to **Public**
5. Click **"Create repository"**

### Step 2 — Upload Files
1. In your new repo, click **"uploading an existing file"**
2. Drag & drop ALL files and folders from this project
3. Commit message: `Initial website launch`
4. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to repo **Settings** → **Pages** (left sidebar)
2. Under **Source**, select `Deploy from a branch`
3. Branch: **main** | Folder: **/ (root)**
4. Click **Save**
5. Wait ~2 minutes → Your site is live at:
   `https://YOUR-USERNAME.github.io/jivan-foundation`

## ✏️ Customisation Checklist

Before going live, update these placeholders in `index.html`:

| Placeholder | Replace With |
|---|---|
| `[ADDRESS]` | Your actual address |
| `[PHONE]` | Your phone number |
| `[EMAIL]` | Your email address |
| `[FACEBOOK_URL]` | Facebook page link |
| `[INSTAGRAM_URL]` | Instagram profile link |
| `[TWITTER_URL]` | Twitter/X profile link |
| `[YOUTUBE_URL]` | YouTube channel link |
| `[WHATSAPP_NUMBER]` | WhatsApp number (with country code, no +) |
| `[DONATION_URL]` | Link to donation platform |

## 🛠️ Adding New Pages

When a page is ready to launch:
1. Open the page file (e.g., `pages/about.html`)
2. Replace the redirect script with the actual page content
3. Keep the same header/footer style as `index.html`
